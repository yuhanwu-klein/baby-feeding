# Quick Start Guide - Cozy Baby Room

Get your 3D cozy baby room running in Blender in 5 minutes!

## Prerequisites

- Blender 2.8 or newer installed
- No additional plugins required
- No external assets needed

## Step-by-Step Instructions

### 1. Open Blender
Launch Blender on your computer

### 2. Open Scripting Workspace
- At the top of the Blender window, click on **"Scripting"** tab
- This will show you a text editor panel

### 3. Load the Script
- In the text editor, click **"Open"** button
- Navigate to where you saved `cozy_baby_room.py`
- Select the file and click **"Open Text Block"**

### 4. Run the Script
- Click the **"Run Script"** button (â–¶ play icon in the text editor)
- Or press `Alt + P` on your keyboard
- Wait 2-3 seconds for the scene to generate

### 5. View Your Room
- Press `Numpad 0` to switch to camera view
- Or navigate to View > Cameras > Active Camera

### 6. Play Animation
- Press `Spacebar` to start the camera animation
- Watch as the camera orbits around the cozy room
- Press `Spacebar` again to pause

### 7. Render an Image
- Press `F12` to render a single beautiful frame
- Wait for rendering to complete
- Press `Esc` to close render window

## Common Issues

**Can't see anything?**
- Press `Home` key to center view on all objects
- Make sure you're in camera view (Numpad 0)

**Script error?**
- Check you're using Blender 2.8+
- Try Window > Toggle System Console to see errors

**Animation not playing?**
- Check the timeline at the bottom
- Make sure frames are set to 1-240
